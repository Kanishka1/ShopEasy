# 🛍️ Full Stack E-Commerce App

A production-ready full stack e-commerce application built with:

## 🚀 Features

- 🔐 Authentication (JWT + Google OAuth)
- 🛒 Cart Management (persistent)
- 💳 Checkout Flow (Address + Payment)
- 📦 Order Management (place, cancel, history)
- 📊 Admin Dashboard (CRUD products)
- 🌙 Dark Mode
- 🔔 Toast Notifications

## 🧱 Tech Stack

### Frontend
- React
- Tailwind CSS
- Context API

### Backend
- Spring Boot
- MongoDB
- GridFS (image storage)
